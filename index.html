NearOS Project

IMPORTANT ! This page present the old version of NearOS, which is deprecated. Please see the new one on my GitHub.

NearOS is a WebOS which permit you to access to a virtual computer everywhere, on every computer and every browser. It is developped in HTML5, CSS3 and JavaScript.

Downloads

You can download the GitHub project or use Git to download it on command-line :

git clone https://github.com/ClementNerma/NearOS-deprecated.git
